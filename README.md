# Three.js + Vite starter kit

This repo is a starting point for every three.js project. 

It include a basic scene with a cube, perspective camera, orbit controls and GUI.


## Installation
Install the dependencies

```bash
npm install
```

## Development

```bash
npm run dev
````

## Build
```bash
npm run build
```
